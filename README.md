# ContactPredictor

TFE "Predicción de contacto efectivo en llamadas telefónicas de cobranza con aprendizaje automático".

Este proyecto compara 25 modelos de ML para optimizar la eficiencia de los call centers de cobranza.

## Simulación de Eficiencia

Los modelos de ML encuentran a los clientes contactables mucho más rápido que las estrategias Aleatoria o Criterio Empresarial.

![simulacion_llamadas](https://github.com/user-attachments/assets/c35f230b-f15d-48ba-827f-72f179b795f2)

## Resultado Principal

El modelo optimizado reduce en un **56.1%** las llamadas necesarias para contactar al 80% de los clientes.

**Stack:** `Python`, `Scikit-learn`, `Pandas`, `XGBoost`, `LightGBM`, `CatBoost`, `Imbalanced-learn`.
