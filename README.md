# ContactPredictor

[cite_start]TFE "Predicción de contacto efectivo en llamadas telefónicas de cobranza con aprendizaje automático"[cite: 1650].

Este proyecto compara 25 modelos de ML para optimizar la eficiencia de los call centers de cobranza.

## Simulación de Eficiencia

[cite_start]Los modelos de ML encuentran a los clientes contactables mucho más rápido que las estrategias Aleatoria o Criterio Empresarial[cite: 2360, 2362].

![simulacion_llamadas](https://github.com/user-attachments/assets/c35f230b-f15d-48ba-827f-72f179b795f2)

## Resultado Principal

[cite_start]El modelo optimizado reduce en un **56.1%** las llamadas necesarias para contactar al 80% de los clientes[cite: 2416].

[cite_start]**Stack:** `Python`, `Scikit-learn`, `Pandas`, `XGBoost`, `LightGBM`, `CatBoost`, `Imbalanced-learn`[cite: 2218].
